HOUSE PRICE PREDICTION

This housing price prediction project involved comprehensive data cleaning, preprocessing, and feature engineering to prepare the dataset for analysis. Key techniques included Exploratory Data Analysis (EDA) to identify influential factors, model building with Linear Regression and performance evaluation using MAPE and R² metrics. GAMES SALES ANALYSIS

The Game Sales SQL project focused on extensive data cleaning, sales performance evaluation, and uncovering industry trends. Missing values in columns like scores, counts, developer, and genre were replaced with placeholders for analysis. Key tasks included identifying top and bottom publishers by global and regional sales, analyzing platform and genre performance, and determining the best-performing games by publisher and year. Market share analysis examined regional contributions to global sales, while rating-based insights highlighted trends in game performance and reviews. The findings provided a comprehensive view of the video game industry's dynamics.

GOOGLE ADVANCED DATA ANALYSIS CAPSTONE PROJECT

The dataset was analyzed to understand its structure, key variables, and missing data. Data preprocessing included cleaning, handling outliers, and feature engineering, followed by exploratory data analysis using Matplotlib and Seaborn to uncover factors influencing employee attrition. Logistic Regression, Decision Trees, and Random Forest models were trained, with Random Forest achieving the best performance based on metrics like accuracy, F1-score, and ROC-AUC. Insights from the model were used to provide actionable recommendations for improving employee retention.
